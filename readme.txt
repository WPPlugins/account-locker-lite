Allows the administrator to lock/ban specific accounts without deleting them.

== Description ==

[Account Locker Lite](http://wordpress.org/plugins/account-locker-lite/) is a state of the art WordPress Plugin which enables administrators to lock user accounts without deleting the users posts and contents.

Locked users will not be able to login or reset their password. All content created by users whose accounts are locked keep accessible by website visitors.

__There is also a [Pro version](https://gewora.net/portfolio-item/account-locker-pro/) available with a lot more features. Make sure to check it out.__

= Main features =
* Lock user accounts via the "Edit User" Screen
* Prohibit users to reset their passwords if their account is locked

== Installation ==

1. Upload `account-locker-lite` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
